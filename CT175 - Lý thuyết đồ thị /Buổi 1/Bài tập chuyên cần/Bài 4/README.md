Cho cấu trúc dữ liệu đồ thị được khai báo sử dụng ma trận ***đỉnh - đỉnh*** như sau:
```
typedef struct {
    int A[100][100];
    int n;
} Graph;
```
Giả sử đồ thị vô hướng, không chứa khuyên, không chứa đa cung

Viết hàm **int deg2(Graph* G)** để đếm số đỉnh có bậc chẵn.
```
int deg2(Graph* G) {
}
```
